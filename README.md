# QoS-Queue-Analyzer
Projekt AiSD: Analiza wydajności kolejki priorytetowej (Kopiec Min-Heap vs Lista)




Jak uruchomić projekt na systemie Linux:

Program w C++ można skompilować za pomocą dostarczonego pliku CMakeLists.txt lub komendą: g++ main.cpp -o projekt

Aby uruchomić skrypt generujący wykresy visualizer.py, wymagane jest zainstalowanie bibliotek Pythona:
pip install pandas matplotlib
