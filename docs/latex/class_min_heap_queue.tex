\doxysubsection{Opis szczegółowy}
Klasa symulująca kolejkę priorytetową dla mechanizmów QoS. 


\begin{DoxyItemize}
\item Wykorzystuje strukturę kopca minimalnego (Min-\/\+Heap) do sortowania przychodzącego ruchu sieciowego. Gwarantuje, że pakiety krytyczne (np. routing, głos) opuszczą interfejs przed ruchem o niskim priorytecie (np. P2P, torrenty). 
\end{DoxyItemize}
